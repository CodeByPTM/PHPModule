Calculator App Algorithm

  1. Start
  2. Display calculator form with two number inputs, operator dropdown, and calculate button
  3. If form is submitted (POST request):
    1. Retrieve num1, num2, and operator from POST data
    2. If operator is '+', set result = num1 + num2
    3. Else if operator is '-', set result = num1 - num2
    4. Else if operator is '*', set result = num1 * num2
    5. Else if operator is '/', check if num2 != 0:
      1. If true, set result = num1 / num2
      2. If false, set result = 'Error: Division by zero'
    6. Else, set result = 'Invalid operator'
  4. Display the result
  5. End
